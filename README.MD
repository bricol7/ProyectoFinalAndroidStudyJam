	DESCRIPTION :

		This is a basic application oriented to students of electrical and electronics, has 5 ways to 
		calculate equivalent circuits, are 5 electric tools useful: a calculator equivalent circuit of 
		resistors in series, in parallel, the equivalent circuit Millman, conversion of delta resistors 
		a star and vice versa. It also has a calculator of the basic parameters present in any electronic 
		circuit: current, resistance and voltage by Ohm's Law.
------------------------------------------------------------------------------------------
	DEVELOPMENT ENVIRONMENT :

		It works on Windows 8.1 Operating System.
		It uses Android Study 1.5.1
		and Java JDK 1.8.0_73
		API15: Android 4.0.3 (IceCreamSandwich)
------------------------------------------------------------------------------------------
	HOW TO USE IT:
------------------------------------------------------------------------------------------
	Steps to calculate sum of resistors in series and parallel:

		1.- Fill in the fields R1, R2 and R3, with resistance values in ?.
		2.- Check the serial or parallel box to obtain the equivalent resistance.
		3.- press the button calculate for the equivalent resistance.
------------------------------------------------------------------------------------------

	Steps for the Delta-Star or Star-Delta conversion:

		1.- Fill the fields of known data [?].
		2.- Select the Delta-Star or Star-Delta box, according to what you need to convert.
		3.- Press the calculate button to convert.
------------------------------------------------------------------------------------------
	Steps to get the Millman`s equivalent circuit: 
	
		1.- Fill the fields of known data.
		2.- Press the calculate button to obtain the values of the equivalent circuit
------------------------------------------------------------------------------------------
	Ohm´s Law calculator:

		1.- Fill the fields of known data
		2.- Select the checkbox current or voltage or resistance, according to what you want to calculate.
		3.- Press the calculate button to obtain the value to obtain the desired parameter.
